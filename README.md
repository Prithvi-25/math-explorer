# Math Explorer V2

An interactive, conceptual math learning platform designed to help students master word problems through visualization and guided hints.

**[Live Demo →](https://prithvi-25.github.io/math-explorer/)**

## New Features (V2)

- **Conceptual Word Problems** — Nuanced problems that require deep understanding, not just number manipulation.
- **Multi-Stage Hints** — Progressive guidance that leads a student to the solution without giving it away immediately.
- **Smart Feedback** — Specific, context-aware responses to common mistakes.
- **Mental Strategy Cards** — Cards that teach different ways to visualize a problem (Number Lines, Bar Models, Area Models).
- **Interactive UI** — Clean, focused learning environment with smooth transitions.

## Built With

- HTML, CSS, JavaScript (Vanilla)
- Google Fonts (Outfit)
- Lucide Icons

## Why I Built This

Most educational math apps focus on drills and speed. I wanted to build something that slows the student down and focuses on the "why" — turning word problems from a source of anxiety into a puzzle to be solved with the right mental strategies.
